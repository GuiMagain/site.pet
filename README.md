# 🐾 Pet&Style - E-commerce de Produtos Pet

O **Pet&Style** é um projeto de estudo focado no desenvolvimento de uma interface moderna e responsiva para uma pet shop. O objetivo principal foi aplicar conceitos avançados de **Tailwind CSS** e estruturação de layouts semânticos.

---

## 🚀 Tecnologias Utilizadas

Para este projeto, utilizei as seguintes ferramentas:

* **HTML5:** Estruturação semântica do conteúdo.
* **Tailwind CSS:** Estilização baseada em utilitários para rapidez e consistência visual.
* **Git & GitHub:** Controle de versão e hospedagem do código.

---

## 🛠️ Funcionalidades do Layout

O site conta com as seguintes seções:

1.  **Banner Principal (Hero):** Chamada de impacto para os produtos.
2.  **Seção de Produtos:** Grid organizado com categorias de rações e brinquedos.
3.  **Depoimentos:** Área de prova social com cards responsivos (ajustados para não "esticarem" em telas grandes).
4.  **Rodapé:** Informações de contato e redes sociais.

## 🎨 Conceitos de Design Aplicados

Como prefiro aprender de forma **visual**, foquei em:
* **Hierarquia Visual:** Uso de pesos de fonte (`font-bold`, `font-light`) para guiar o olhar.
* **Espaçamento:** Utilização sistemática de `p-` (padding) e `m-` (margin) para evitar que o conteúdo pareça "apertado".
* **Responsividade:** Uso de prefixos `md:` e `lg:` para garantir que o site fique bonito no celular e no PC.
